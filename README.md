# html-fundamentals-course-nextech

Repositorio del curso **Fundamentos de HTML y CSS para Principiantes**, dirigido a estudiantes de secundaria sin experiencia previa en desarrollo web.

El curso explica que es un sitio web, como se estructura con HTML y como se aplica diseno visual con CSS. No requiere instalaciones: todas las practicas estan pensadas para realizarse desde el navegador con editores online como CodePen, JSFiddle o Replit.

## Documento principal

- [Plan de curso en Markdown](./course-plan.md)
- [Guion completo de videos](./scripts/full-video-script.md)

## Objetivo del curso

Que el estudiante pueda crear una pagina web basica usando HTML para organizar el contenido y CSS para mejorar su presentacion visual.
