# Lección 3: Tags Elements Attributes

**Duración recomendada:** 7-8 minutos

## Objetivo

Explicar el tema de la lección con ejemplos sencillos y una práctica corta.

## Guion base

Hola a todos. Bienvenidos a esta lección del curso de fundamentos de HTML y CSS.

En esta clase veremos el tema correspondiente paso a paso, usando ejemplos simples y ejecutando el resultado en el navegador.

Primero revisaremos el código inicial en la carpeta `starter-files`. Luego completaremos el ejemplo y compararemos el resultado con el archivo de `completed-examples`.

## Práctica

Abre el archivo starter de esta lección, modifica el contenido solicitado y visualiza el resultado en el navegador.

## Cierre

En esta lección aprendimos un concepto clave para construir páginas web. En la siguiente clase continuaremos agregando más estructura y estilo.
