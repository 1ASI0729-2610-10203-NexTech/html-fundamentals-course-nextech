# Leccion 8: Recomendaciones y errores comunes

## Objetivo

Que el estudiante revise su pagina final, identifique errores comunes y conozca recomendaciones para seguir aprendiendo HTML y CSS.

## Duracion

5 minutos

## Guion resumido

En esta leccion se repasan consejos finales para principiantes. El objetivo es que el estudiante entienda que cometer errores es parte normal del aprendizaje y que revisar el codigo paso a paso ayuda a encontrar problemas.

## Errores comunes

- No cerrar etiquetas HTML.
- Escribir atributos sin comillas.
- Olvidar el atributo `alt` en imagenes.
- Colocar codigo CSS dentro de la zona HTML.
- No probar enlaces o imagenes.
- Escribir propiedades CSS sin punto y coma.

## Practica final

Revisar la pagina de perfil creada en la leccion 7 y confirmar:

- El titulo aparece correctamente.
- La imagen carga.
- La lista se ve ordenada.
- El enlace funciona.
- Los estilos CSS se aplican.

## Cierre

HTML y CSS son la base del desarrollo web. Despues de este curso, el estudiante puede seguir practicando con paginas mas completas, diseno responsive y JavaScript.
